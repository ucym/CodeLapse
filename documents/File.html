seek( int   $offset, int   $whence = self::SEEK_SET, ) : boolean

Description

ファイルポインタを移動します。 (http://www.php.net/manual/ja/function.fseek.php)
whenceの値について - whence の値は以下のようになります。 - File::SEEK_SET - 位置を offset バイト目に設定する - File::SEEK_ADD - 現在の位置に offset を加えた位置に設定する - File::SEEK_END - ファイル終端に offset を加えた位置に設定する - ファイルの終端から数えた位置に移動するには、$whenceを File::SEEK_END として、負の値をoffsetに渡す方法があります。

Arguments

Name Type Description Default
$offset int

ファイルポインタの位置。
$whence int

ファイルポインタの位置の計算方法。

 self::SEEK_SET

Return value

Type Description
boolean 成功すると TRUE を返し、そうでなければ FALSE を返します。
