\Roose_DB

MySQL接続ユーティリティ

このクラスはRooseライブラリ環境下（bootstrap.phpを読み込んだ状態）では "DB"クラスとして利用可能です。 単体ライブラリとして利用しているときは Roose_DBがクラス名になります。

//-- Roose(bootstrap.php)を読み込んだ時
require 'pathToRoose/bootstrap.php';
$result = DB::query('SOME SQL QUERY');

//-- db.phpを単体で読み込んだ時
require 'pathToDb/db.php';
$con = Roose_DB::instance('hostName', 'user', 'password');
$result = $con->query('SOME SQL QUERY');

データベース接続からSQL送信までの流れ

db.php を単体で読み込んだ時

// （例: userid => lzm, password => wox#a'zlp）
$userid = $_POST['userid'];
$password = $_POST['password'];

// DBと接続する
$con = Roose_DB::instance('localhost', 'user', 'password');

// クエリを送る
// 配列に渡した値が、クエリに埋め込まれていることに注目。
// （送信されるクエリ: SELECT * FROM Users WHERE userid = 'lzm' AND password = 'wox#a\'zlp'）
$result = $con->query(
     'SELECT * FROM Users WHERE userid = :id AND password = :pass',
     array(':id' => $userid, ':pass' => $password)
);

if ($result->fetch() !== false) {
     echo 'ログイン成功';
} else {
     echo 'ログイン失敗';
}

bootstrap.phpを取り込んだ時

//!! （事前に "config/db.php"を設定しておく必要があります。）

// （例: userid => lzm, password => wox#a'zlp）
$userid = Input::post('userid');
$password = Input::post('password');

// クエリを送る
// 配列に渡した値が、クエリに埋め込まれていることに注目。
// （送信されるクエリ: SELECT * FROM Users WHERE userid = 'lzm' AND password = 'wox#a\'zlp'）
$result = DB::query(
     'SELECT * FROM Users WHERE userid = :id AND password = :pass',
     array(':id' => $userid, ':pass' => $password)
);

if ($result->fetch() !== false) {
     echo 'ログイン成功';
} else {
     echo 'ログイン失敗';
}

取得した結果を全件表示する

//-- Rooseライブラリ使用中の場合を想定
// SQLの実行結果は $result に入っている
foreach ($result as $row) {
     // $rowに一行分のデータが入ってくる
     print_r($row);
}

instance()

instance(string $connection_name, null|string $user, null|string $password, null|boolean $newConnection) : \DBConnection

指定した接続のコネクションインスタンスを取得します。

パラメータを二つ以上指定した時、
最初のパラメーターを接続先ホストとして
続く２つのパラメーターをユーザー名、パスワードとして、データベースに接続します。

パラメータが一つしか指定されない場合、
接続設定は config/db.phpの設定を参照します。
config/db.php に接続設定を記述するか
Configクラスを利用して、db名前空間に接続設定を読み込んでください。

単体ライブラリとして利用する場合、ホスト名・ユーザーは必ず指定する必要があります。

Parameters

string $connection_name

使用する接続名
null|string $user

(optional) ユーザー名
null|string $password

(optional) パスワード
null|boolean $newConnection

(optional) 接続を新規生成するか指定します

Throws

\OutOfBoundsException

dbconfig.phpに記述されていないコネクション名が指定された時にスローされます。

\InvalidArgumentException

単体ライブラリとして利用しているときに、ホスト名とユーザー名が指定されない時に発生します。

Returns

\DBConnection —

DBConnectionインスタンス
