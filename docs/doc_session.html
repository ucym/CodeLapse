セッションclass Session

$_SESSION変数の操作を行うクラスです。

セッション操作の例

<?php
// セッションへ値を設定する
Session::set('auth.user_id', $user_id);

// セッションから値を取得する
// （$_SESSION['auth']['user_id']が設定されていない場合、nullを返す）
$user_id = Session::get('auth.user_id');

// （$_SESSION['auth']['user_id']が設定されていない場合、falseを返させる）
$user_id = Session::get('auth.user_id', false);

通常、セッションを利用する前にはsession_startをコールする必要がありますが
CodeLapseにおいては、Sessionクラスが自動的に初期化を行います。
